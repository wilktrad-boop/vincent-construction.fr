## Identifier les petits vers noirs dans les toilettes

Vous avez remarqué de petits vers noirs ou gris foncé dans la cuvette des WC, sur les parois ou autour de la base ? Pas de panique. Dans la grande majorité des cas, il s'agit de **larves de psychodidae**, plus connues sous le nom de mouches papillons ou mouches des éviers.

### À quoi ressemblent-ils ?

- **Taille** : 4 à 10 mm de longueur
- **Couleur** : noir, gris foncé ou brun
- **Forme** : allongée, légèrement aplatie
- **Mouvement** : lent, ondulant

Ces larves ne sont pas des vers à proprement parler, mais bien des larves d'insectes. Elles sont inoffensives pour la santé, mais leur présence indique un problème d'hygiène dans les canalisations.

## Pourquoi apparaissent-ils ?

Les larves de mouches papillons se développent dans les **dépôts organiques** qui tapissent l'intérieur des canalisations : résidus de savon, cheveux, matières grasses, biofilm bactérien... Les WC peu utilisés sont particulièrement concernés.

### Les causes principales

1. **WC secondaires peu utilisés** : l'eau stagnante et le manque de flux créent un environnement idéal
2. **Joint de cuvette défectueux** : si le joint d'étanchéité entre la cuvette et la bride d'évacuation est abîmé, les insectes remontent par cette voie
3. **Canalisations encrassées** : des dépôts épais dans les tuyaux offrent un milieu de ponte parfait
4. **Siphon désamorcé** : si le siphon est sec (évaporation dans un WC inutilisé), les insectes accèdent librement depuis les canalisations

## Comment s'en débarrasser

### Action immédiate : le nettoyage mécanique

1. **Nettoyez la cuvette** en profondeur avec un produit WC et une brosse
2. **Versez de l'eau bouillante** dans la cuvette pour tuer les larves dans le siphon
3. **Nettoyez le dessous de la lunette** et les recoins souvent oubliés
4. **Vérifiez le joint de base** : si vous voyez de l'humidité autour du pied de la cuvette, le joint est probablement à remplacer

### Traitement des canalisations

Pour éliminer le biofilm où pondent les mouches :

- **Bicarbonate + vinaigre blanc** : versez un demi-verre de bicarbonate dans la cuvette, ajoutez un verre de vinaigre blanc, laissez agir 30 minutes puis tirez la chasse avec de l'eau chaude
- **Gel déboucheur enzymatique** : plus efficace qu'un déboucheur chimique car il s'attaque spécifiquement aux dépôts organiques sans abîmer les tuyaux
- **Eau de Javel diluée** : efficace mais à utiliser ponctuellement (agressive pour les joints et mauvaise pour les fosses septiques)

### Prévention à long terme

- **Tirez la chasse quotidiennement**, même dans les WC peu utilisés
- **Aérez la pièce** : les mouches papillons aiment l'humidité stagnante
- **Nettoyez régulièrement** le siphon avec du vinaigre blanc (1 fois par mois)
- **Vérifiez l'étanchéité** : un joint de cuvette défectueux doit être remplacé

## Quand faire appel à un professionnel ?

Si les vers persistent malgré un nettoyage approfondi, le problème est peut-être plus profond :

- **Canalisation fissurée** dans le mur ou le sol
- **Défaut de ventilation** du réseau d'évacuation
- **Problème de pente** : une canalisation avec une pente insuffisante accumule les dépôts

Un plombier pourra réaliser une inspection caméra de vos canalisations pour identifier la source exacte du problème.

## Ce ne sont pas des vers noirs ?

D'autres organismes peuvent ressembler à des vers dans les toilettes :

- **Larves de moustiques** : fines, transparentes, se tortillent vivement dans l'eau
- **Asticots** : blancs ou crème, plutôt dans les poubelles que les WC
- **Vers de drain** (Hermetia illucens) : plus gros, segmentés, fréquents dans les régions chaudes

Dans tous les cas, le traitement est similaire : nettoyage mécanique des canalisations, suppression des dépôts organiques et maintien d'une bonne hygiène.
