## Le principe du WC avec lave-main

Le concept est simple : un petit lavabo est intégré au-dessus du réservoir de chasse d'eau. Quand vous vous lavez les mains, l'eau savonneuse s'écoule dans le réservoir et sera utilisée pour la prochaine chasse. C'est à la fois un gain de place et une économie d'eau.

Ce type de WC existe en deux versions :
- **Le réservoir avec lave-main intégré** : il se pose sur un WC existant en remplacement du couvercle de réservoir
- **Le WC complet avec lave-main** : un bloc monolithique conçu d'usine

## Les avantages

### Gain de place considérable
Dans un WC de 1 à 2 m², il n'y a souvent pas la place pour un lave-main séparé. Le WC avec lave-main intégré résout ce problème sans empiéter sur l'espace de circulation. C'est son argument numéro un.

### Économie d'eau
L'eau utilisée pour le lavage des mains remplit le réservoir. Sur une année, une famille de 4 personnes peut économiser entre 5 000 et 7 000 litres d'eau. Ce n'est pas négligeable sur la facture.

### Installation simplifiée
Pas besoin de tirer une évacuation supplémentaire ni de raccorder un lave-main au réseau. L'arrivée d'eau du WC alimente directement le lave-main. En rénovation, c'est un avantage majeur.

### Hygiène encouragée
Le lave-main est juste là, impossible de l'ignorer. Le geste de se laver les mains devient automatique, ce qui améliore l'hygiène globale du foyer.

## Les inconvénients

### Confort d'utilisation limité
Le lave-main intégré est petit. Se laver correctement les mains dans une vasque de 20 cm de diamètre n'est pas très confortable. Les éclaboussures sont fréquentes, surtout pour les enfants ou les personnes de grande taille.

### Eau froide uniquement
La plupart des modèles ne proposent que de l'eau froide, puisqu'ils sont raccordés sur l'arrivée d'eau du WC. En hiver, se laver les mains à l'eau glacée n'est pas agréable. Certains modèles haut de gamme intègrent un petit chauffe-eau instantané, mais le prix grimpe.

### Nettoyage contraignant
Le lave-main au-dessus du réservoir est difficile à nettoyer proprement. L'accès est restreint, et les dépôts de savon s'accumulent dans des recoins peu accessibles. L'eau savonneuse dans le réservoir peut aussi favoriser l'encrassement du mécanisme de chasse.

### Esthétique discutable
Soyons honnêtes : un lave-main au-dessus du réservoir de WC, ce n'est pas le summum de l'élégance. Les modèles japonais sont bien conçus, mais les versions européennes d'entrée de gamme peuvent faire "bricolage".

### Incompatible avec certains WC
Les WC suspendus (avec bâti-support encastré) ne peuvent pas recevoir de lave-main intégré, puisque le réservoir est dans le mur. C'est une solution réservée aux WC posés avec réservoir apparent.

## Pour qui est-ce adapté ?

**Bonne idée pour :**
- Les toilettes séparées de moins de 2 m² sans point d'eau
- Les WC d'appoint (sous-sol, garage aménagé)
- Les petits appartements où chaque centimètre compte
- Les locations où les travaux sont limités

**Mauvaise idée pour :**
- Les WC suspendus (pas compatible)
- Les familles avec jeunes enfants (inconfortable, éclaboussures)
- Les personnes qui recherchent un confort optimal
- Les WC assez grands pour accueillir un vrai lave-main

## Prix indicatifs

| Type | Prix |
|------|------|
| Couvercle lave-main adaptable | 80-150 euros |
| WC complet avec lave-main (entrée de gamme) | 200-350 euros |
| WC complet design (type japonais) | 500-1 200 euros |
| Kit adaptable avec mitigeur | 120-250 euros |

Le couvercle adaptable est la solution la plus économique pour tester le concept sans remplacer votre WC existant.

## Notre avis

Le WC avec lave-main intégré est une **solution pragmatique pour les petits espaces**. Il remplit son rôle principal (permettre le lavage des mains dans un WC sans point d'eau) de manière efficace. L'économie d'eau est un bonus appréciable.

Mais ne vous attendez pas au même confort qu'un vrai lavabo. Si vous avez la place pour un lave-main mural compact (il en existe à partir de 25 cm de profondeur), c'est une meilleure solution à long terme.
